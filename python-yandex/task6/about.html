О гороскопе

Сайт содержит ежедневные гороскопы для всех знаков зодиака. Гороскопы генерируются случайно с использованием цепей маркова. Выборка для построения цепей маркова производилась здесь. Ваш гороскоп может содержать ненормативную лексику, не соответствовать действительности и быть шизофазией
